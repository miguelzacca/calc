# calc

Simple command line calc with Node.js and mathjs.

## Install - Linux

```bash
git clone https://github.com/miguelzacca/calc.git
chmod +x ~/calc/config.sh ~/calc/src/index.js
~/calc/config.sh
```

## Install - Windows

```bash
git clone https://github.com/miguelzacca/calc.git
%HOMEPATH%/calc/config.bat
```

## Use

```bash
calc <EXPRESSION>
```

## Example

```bash
calc 10+9*(2/3)-87
```

output:

```txt
-71
```
